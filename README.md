# 🚀 Hello, Backboard Challenge

This project demonstrates how to use the **Backboard.io** unified AI stack to create a stateful assistant.

## 🛠️ Features
- **Assistant Initialization:** Set up an AI agent with specific behavioral rules.
- **Thread Management:** Created a persistent conversation session ID.
- **Message Requests:** Implemented non-streaming communication to get AI responses.

## 🧰 Tech Stack
- **Language:** Python 3.12
- **Infrastructure:** Backboard.io SDK
- **Environment:** GitHub Codespaces

## 🚀 How to Run
1. Install the SDK: `pip install backboard-sdk`
2. Run the script: `python app.py`

## 🏁 Result
The implementation follows the 3-step workflow (Assistant -> Thread -> Message) to build a controlled AI environment similar to a custom ChatGPT.
